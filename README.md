# Agromatic Website

GitHub-Pages-kompatible Entwicklerstruktur.

## Struktur

```text
index.html
produkte.html
anwendungen.html
atex.html
unternehmen.html
kontakt.html
assets/
  css/
    main.css
    header.css
    hero.css
    content.css
    footer.css
  js/
    main.js
  img/
    hero-bg.png
    logo.png
    prozessumgebung.png
    produkt-detail.png
    produkte/
```

## Upload

Den kompletten Inhalt dieses Ordners in das GitHub-Repository kopieren, committen und pushen. GitHub Pages kann direkt aus `main` und `/root` bereitstellen.


## Produkt-Unterseiten
- dreh-schwenkantriebe.html
- linearantriebe.html

Die Karten auf `produkte.html` verlinken direkt auf diese Unterseiten.
