Agromatic Konfigurator Teststand

Diese ZIP ist so aufgebaut, dass die enthaltenen Dateien direkt in den Repository-Ordner kopiert werden können.
Vorhandene Dateien mit gleichem Namen überschreiben.

Geänderte/neue Dateien:
- dreh-schwenkantriebe.html

Testregeln:
- 100 Nm = kleine Haube
- 120 Nm = kleine Haube mit Ring
- 130 Nm = große Haube
- 140 Nm = große Haube mit Ring
- Wellenvarianten: ohne Welle, Innenvierkant, Federwelle
- Zusatzgetriebe testweise auswählbar
- Handrad-Varianten noch nicht aktiv

Nach dem Kopieren:
1. In GitHub Desktop Änderungen prüfen
2. Commit erstellen
3. Push ausführen
4. Website im Browser mit STRG+F5 neu laden
